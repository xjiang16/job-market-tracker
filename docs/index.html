What do data engineering postings actually ask for?

A pipeline that pulls live postings from the Adzuna API, lands them in Snowflake, and mines them with dbt — orchestrated with Airflow and refreshed automatically every day via a scheduled GitHub Actions workflow. Below is a real query against the current dataset.

analytics.job_skills 
SELECT skill, COUNT(*) AS mentions
FROM analytics.job_skills
WHERE mentions_python OR mentions_sql
   OR mentions_airflow OR mentions_snowflake OR mentions_dbt
GROUP BY skill
ORDER BY mentions DESC;

How it works

01

INGEST

A Python script pulls postings from the Adzuna API across multiple job titles and locations.

02

LOAD

Raw postings land in Snowflake as an append-only log — nothing is cleaned or deduplicated yet.

03

TRANSFORM

dbt deduplicates postings, extracts skill mentions, and runs automated data-quality tests.

04

ORCHESTRATE

Airflow runs the full pipeline locally in dependency order; a scheduled GitHub Actions workflow refreshes the public snapshot daily.
